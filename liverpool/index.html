Liverpool Council Spending

An Open Innovations project to visualise public body spending. Liverpool only publish data in XLS and PDF format. So we rely on processing and cleaning by North West Open Data. They have processed transactions over £500 for 2019 for Liverpool Council (except for items which are commercially sensitive or for which there may be data protection issues). This visualisation is a work-in-progress but let us know if it is useful.
